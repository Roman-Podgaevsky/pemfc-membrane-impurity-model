# Описание проекта для анкеты ШАД

## Название

**MEMB impurity model: модель переноса примесей через мембрану PEMFC**

## Краткое описание

Проект посвящён инженерному моделированию переноса компонентов через мембрану водородного топливного элемента PEMFC. В Python реализована учебно-исследовательская модель, которая рассчитывает перенос `H2`, `N2`, `O2` и `H2O` через мембрану, реакционный расход водорода и накопление примесей в анодном контуре при закрытой рециркуляции.

Модель построена по открытым литературным источникам по PEMFC-мембранам. Формат входных данных и выходных таблиц сделан близким к инженерной модели в Simcenter Amesim, что позволяет сопоставлять Python-расчёт с CAE-моделью.

## Зачем был сделан проект

Цель проекта: показать, как инженерная CAE-задача может быть формализована в виде воспроизводимого Python-ноутбука.

Практический контекст: водородный топливный элемент, анодная рециркуляция, перенос примесей через мембрану, контроль состава газа и оценка момента продувки анодного контура.

## Что реализовано

- расчёт активности воды на анодной и катодной сторонах;
- расчёт гидратации мембраны;
- расчёт объёмной доли воды в мембране;
- расчёт electro-osmotic drag;
- расчёт диффузионного переноса воды;
- расчёт crossover `O2`, `N2`, `H2`;
- расчёт расхода водорода по закону Фарадея;
- вывод компонентных молярных потоков;
- расчёт накопления примесей при закрытой рециркуляции;
- документация с пояснением уравнений и источников.

## Навыки, которые показывает проект

- Python для инженерных расчётов;
- работа с физическими моделями;
- структурирование расчётного notebook;
- валидация инженерной модели по CAE-результатам;
- аккуратное описание допущений и области применимости;
- перевод инженерной логики из CAE-среды в воспроизводимый код;
- подготовка технического репозитория для GitHub.

## Почему это релевантно ШАД

Проект находится на стыке инженерного моделирования, анализа данных и воспроизводимых вычислений. В нём есть постановка задачи, математическая модель, программная реализация, табличные результаты и проверка на инженерном референсе.

Для обучения в ШАД это показывает интерес к прикладным задачам, где Python используется не только для обработки данных, но и для построения проверяемых вычислительных моделей.
