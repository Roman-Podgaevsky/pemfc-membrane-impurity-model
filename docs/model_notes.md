# Пояснения к модели

## 1. Физический объект

Рассматривается мембрана PEMFC между анодной и катодной газовыми полостями.

Через мембрану учитываются:

- перенос кислорода из катода в анод;
- перенос азота из катода в анод;
- перенос водорода из анода в катод;
- перенос воды за счёт диффузии;
- перенос воды за счёт electro-osmotic drag.

Также учитывается реакционный расход водорода на аноде.

## 2. Входные данные

Входные данные заданы в стиле инженерной CAE-модели:

- температура анода и катода;
- абсолютное давление анода и катода;
- молярные доли `H2`, `N2`, `H2O`, `O2`;
- ток БТЭ;
- число ячеек;
- активная площадь;
- толщина мембраны;
- параметры материала мембраны.

## 3. Основные расчётные блоки

### 3.1. Давление насыщенного пара

Давление насыщенного пара воды используется для расчёта активности воды:

```text
a_w = p_H2O / p_sat(T)
```

### 3.2. Гидратация мембраны

Водосодержание мембраны задаётся как функция активности воды:

```text
lambda_mem = f(a_w)
```

Эта зависимость нужна для расчёта переноса воды и газовых пермеансов.

### 3.3. Gas crossover

Для газов используется инженерная форма:

```text
n_i = k_i * Δp_i
```

где `k_i` зависит от состояния мембраны, а `Δp_i` задаётся разностью парциальных давлений на двух сторонах мембраны.

### 3.4. Перенос воды

Суммарный поток воды складывается из двух частей:

```text
n_H2O_net = n_H2O_diff - n_H2O_drag
```

Диффузия воды определяется градиентом водосодержания мембраны, а electro-osmotic drag связан с протонным током.

### 3.5. Расход водорода

Реакционный расход водорода рассчитывается по закону Фарадея:

```text
n_H2_cons = - I * Ncell / (2F)
```

Минус отражает выбранную знаковую конвенцию: водород расходуется на аноде.

## 4. Блок закрытой рециркуляции

В блоке закрытой рециркуляции оценивается скорость накопления примесей при закрытых клапанах продувки.

Учтены:

- восполнение реакционного расхода чистым водородом;
- требуемый входной расход через стехиометрическое число;
- накопление `N2` и `O2`;
- перенос воды через мембрану;
- снижение влажности после влагоотделителя.

## 5. Ограничения

Модель является компактной 0D/1D-инженерной оценкой.

Не учитываются:

- распределённые поля концентраций по каналам;
- двухфазная гидродинамика;
- локальная электрохимия;
- деградация мембраны;
- детальная динамика объёмов анодного контура;
- полный контроллер продувки и дренажа.

## 6. Назначение

Модель подходит для:

- учебной демонстрации;
- проверки порядков величин;
- подготовки расчётной логики для CAE;
- сопоставления Python-расчёта с Amesim;
- объяснения инженерной постановки задачи в портфолио.
