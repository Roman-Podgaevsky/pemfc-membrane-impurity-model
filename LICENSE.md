# Лицензия и назначение

Материалы репозитория предназначены для учебного и исследовательского использования.

Разрешается:

- читать и запускать код;
- использовать фрагменты для обучения;
- адаптировать модель под собственные исследовательские задачи;
- ссылаться на репозиторий как на учебно-инженерный пример.

Ограничения:

- проект не является официальной моделью Simcenter Amesim;
- проект не содержит заявлений о полном совпадении с коммерческими CAE-моделями;
- результаты требуют инженерной проверки перед применением в расчётах реальных изделий.

Рекомендуемая формулировка при использовании:

> Учебно-исследовательская Python-модель переноса компонентов через мембрану PEMFC, построенная по открытым источникам и сопоставленная с инженерной моделью Amesim.
